# ACCESS
```
provide the base access interfaces requried for any sort of web service.
the abstract principal is the base principal for the web service that can be implemented and 
initated at the web service level. the access service is an abstract service that provides a 
resource access and CRUD level of access to a certain resource, it is up to the implementor how to define 
the access logic.
```
